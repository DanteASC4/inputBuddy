## Data Stored by InputBuddy

This extension saves a few things:

- Label & Answer pairs entered by you the user
- Settings configuration
- Profile names are also saved if utilized

Do not use InputBuddy to store highly sensitive information.

## Website Access

InputBuddy interacts with web pages you visit to automatically fill out fields. On supported websites, the following elements may be processed locally

- text input fields
- text near input fields
- placeholder text
- labels
- input metadata

This is needed for finding relevant text field identifiers for auto-filling even on sites with complex or custom internal structures.

This website data is processed locally in your browser and is not sent to the developer or any third party.

## Data Storage

All data for the extension is stored within your local browser. Nothing is sent anywhere external, meaning if you opened the extension on a different device you'd be met with a blank slate.

Nothing is synced with any external server or resource.

If this data was not stored, you would need to set everything up again each time you use attempt to use the extension.

## Data Sharing

InputBuddy does not sell, share, or transmit your saved data to third parties.

## Deleting Your Data

You can delete saved answers and profiles inside the extension. You can also remove all locally stored data by uninstalling the extension or clearing the extension's browser storage.

## Telemetry

InputBuddy does not currently collect telemetry or analytics.

If telemetry is added in a future version, this policy will be updated before release and the extension listing will disclose what is collected.

## Security Note

InputBuddy is intended for common form answers. Do not use it to store passwords, PINs, security answers, payment card numbers, or other highly sensitive credentials.
